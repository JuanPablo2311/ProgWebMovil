HIPERTEXTO

es una estructura no secuencial que permite crear, agregar, enlazar y compartir información de diversas fuentes

ETIQUETAS HTML

Son indicadores del lenguaje de marcas de hipertexto (o HTML, por sus siglas en inglés) que le dan formato y estructura al contenido de las páginas web.

ELEMENTO HTML

Es un tipo de componente de documento de HTML, uno de los diversos tipos de nodos de HTML

DOCTYPE

Informa al navegador que versión de HTML (o XML) se usó para escribir el documento.

CHARSET

Define que tipo de caracteres vamos a utilizar en la web, para que el navegador lo detecte y lo visualice correctamente.

ARTICLE

Es un elemento semántico. Se usa más comúnmente para contener información que puede distribuirse independientemente en el sitio que aparece

FOOTER

Representa un pie de página para el contenido de sección más cercano o el elemento raíz de sección

SPAN

Es un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea.

Formularios web

Interfaz que permite a los usuarios la introducción de datos que generalmente se envían a un servidor

CSS

En español «Hojas de estilo en cascada», es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado

Front-End

Consiste en la conversión de datos en una interfaz gráfica para usuarios

ETIQUETA DE BLOQUE

Las etiquetas de bloque permiten definir bloques de contenido coherente, formado por texto e imágenes.

ETIQUETA DE LINEA

Una etiqueta de línea es aquella que ocupa el espacio mínimo necesario en horizontal, y permite que otro elemento se coloque a su lado

ARQUITECTURA WEB

Es la planificación y el diseño de los componentes técnicos, funcionales y visuales de un sitio web, antes de que sea diseñado, desarrollado e implementado

INTERNET

Es un conjunto descentralizado de redes de comunicaciones interconectadas, que utilizan la familia de protocolos TCP/IP

Servidor

es un conjunto de computadoras capaz de atender las peticiones de un cliente y devolverle una respuesta en concordancia
